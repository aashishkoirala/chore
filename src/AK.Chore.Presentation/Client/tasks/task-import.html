Paste the task data in the area below and hit Import. There should be one line per task with the first line having field names. All fields should be separated by TABs.

The following are fields you can use:
  • Description: What the task actually is; required.
  • Folder: The full path of the folder (as it appears above the task description in the task list, e.g. "Personal/My Folder") where the task goes; required.
  • Finish By: When to finish by, can include just a date (YYYY-MM-DD) or a date and time (YYYY-MM-DD hh:mm); required for non-recurring tasks.
  • Start By: When to start by, can include just a date (YYYY-MM-DD) or a date and time (YYYY-MM-DD hh:mm); optional.
  • Recurring: Whether the task is recurring, can be "Yes" or "No"; optional - defaults to "No".
  • Recurrence Type: Type of recurrence, can be "Hourly", "Daily", "Weekly", "Monthly" or "Yearly"; required for recurring tasks.
  • Recurrence Interval: Number of intervals to recur in (e.g. Hourly with 2 means recur every 2 hours); optional, defaults to 1 for recurring tasks.
  • Recurrence Duration: The duration of the task in hh:mm format; optional, defaults to 01:00 (i.e. 1 hour) for recurring tasks.
  • Recurrence Time of Day: The time of day when the task recurs; optional.
  • Recurrence Day of Month: The day of month when the task recurs; optional.
  • Recurrence Days of Week: For weekly recurrence, the days of week when the task recurs, comma separated list of days of the week, e.g. "Sunday,Monday,Thursday"; optional.
  • Recurrence Month of Year: The month of year when the task recurs, can be one of the months, e.g. January, February etc.; optional.
