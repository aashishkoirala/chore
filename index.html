CHORE

  • Go to Application
  • Go to GitHub Repository

CHORE is a task management application that I built for personal use. Hopefully it comes in handy for you too. It lets you create and track tasks based on deadlines or recurrence, organize tasks into folders, use built-in or custom filters to look for tasks, get a weekly calendar view based on your tasks, export and import tasks to/from flat data among other things.

Features
The initial version consists of the following features:
  • Create and track tasks based on deadlines or recurrence.
  • Organize tasks into folders.
  • Use built-in or custom filters to look for tasks.
  • Get a weekly calendar view based on your tasks.
  • Export and import tasks to/from flat data.
  • Export and import all user data.
